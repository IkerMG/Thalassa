# Integración: Comunicación Java <-> Python

## Protocolo
La comunicación será **Síncrona** mediante **REST (HTTP/JSON)**.

## Flujo de Trabajo (Workflow)
1. El Usuario pulsa "Buscar en tiendas" en el Frontend (React).
2. **React** envía la petición al Controller de **Spring Boot**.
3. **Spring Boot** actúa como cliente:
   - Usa `WebClient` (reactivo) o `RestTemplate` para llamar a `http://localhost:8000/scrape`.
4. **FastAPI (Python)** procesa y responde.
5. **Spring Boot** recibe el JSON, lo mapea a un DTO (Data Transfer Object) y lo envía de vuelta al **Frontend**.

## Seguridad Interna
- El microservicio de Python solo aceptará peticiones provenientes de la IP del servidor de Java (Whitelist) para evitar accesos externos directos.
