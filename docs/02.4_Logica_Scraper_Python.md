# Lógica del Microservicio Scraper (Python + FastAPI)

## Objetivo
Obtener en tiempo real el precio, nombre y foto de productos de acuariofilia de tiendas referencia (ej: Tiendanimal, Kiwoko, CoralReef).

## Stack Tecnológico
- **Framework:** FastAPI (por su velocidad y documentación Swagger automática).
- **Librerías:** - `requests`: Para realizar las peticiones HTTP a las webs.
  - `BeautifulSoup4`: Para parsear el HTML y extraer los selectores CSS.
  - `fake-useragent`: Para evitar bloqueos simulando ser un navegador real.

## Proceso de Scraping
1. **Recepción:** Java envía una `keyword` (ej: "Skimmer Tunze 9004").
2. **Búsqueda:** Python construye la URL de búsqueda de la tienda seleccionada.
3. **Extracción (Parsing):**
   - Localiza el contenedor del producto en el HTML.
   - Extrae el texto del precio (limpiando símbolos de moneda).
   - Extrae el atributo `src` de la etiqueta `<img>`.
4. **Respuesta:** Devuelve un objeto JSON estructurado al backend de Java.

## Gestión de Errores
- Si la web tarda > 5s, devuelve un `TimeoutError`.
- Si el selector CSS ha cambiado, devuelve un `ParsingError` para que Java use datos por defecto.
