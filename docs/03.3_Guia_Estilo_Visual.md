- **Color Primario (Deep Sea):** `#002B5B` (Azul muy oscuro para barras de navegación).
    
- **Color Secundario (Cyan Wave):** `#2B7A0B` o mejor un `#0078AA` (Azul vibrante para botones de acción).
    
- **Color de Fondo (Soft Foam):** `#F8F9FA` (Gris casi blanco para que la vista no se canse).
    
- **Color de Alerta (Coral Red):** `#FF6B6B` (Para avisos de incompatibilidad o parámetros fuera de rango).
    
- **Tipografía:** _Montserrat_ (para títulos, moderna y limpia) y _Open Sans_ (para textos largos, legible).
