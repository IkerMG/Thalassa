# 02.3 API Endpoints

> **🔒 Nota de Seguridad (JWT):** Todos los endpoints que requieran el rol `USER` o `ADMIN` deben incluir obligatoriamente la cabecera HTTP: `Authorization: Bearer <token_jwt>`. Si no se incluye, el servidor devolverá `401 Unauthorized` o `403 Forbidden`.

### ☕ Microservicio A: Java Spring Boot (Puerto 8080)

| Método | Endpoint | Descripción | Roles |
|---|---|---|---|
| **POST** | `/api/auth/register` | Registro de nuevo usuario. | Público |
| **POST** | `/api/auth/login` | Login y obtención de Token JWT. | Público |
| **GET** | `/api/aquariums` | Lista todos los acuarios del usuario. | USER |
| **POST** | `/api/aquariums` | Crea un nuevo proyecto de acuario. | USER |
| **GET** | `/api/aquariums/{id}` | Detalle completo: equipo + fauna. | USER |
| **GET** | `/api/wishlist` | Lista de deseos del usuario. | USER |
| **POST** | `/api/wishlist/search` | Trigger: Llama a Python para buscar. | USER |
| **DELETE** | `/api/livestock/{id}` | Eliminar pez/coral (baja o venta). | USER |

### 🐍 Microservicio B: Python FastAPI (Puerto 8000)

| Método | Endpoint | Parámetros | Respuesta JSON |
|---|---|---|---|
| **GET** | `/scrape/price` | `keyword` (str) | `{name, price, img, url, store}` |
| **GET** | `/health` | Ninguno | `{status: "up"}` |

---

**⚙️ Nota Técnica de Integración:** El endpoint de Java `/api/wishlist/search` no hace el scraping por sí mismo. En su lugar, hace una petición interna al microservicio de Python usando `WebClient` o `RestTemplate`, y devuelve el resultado consolidado al Frontend.
